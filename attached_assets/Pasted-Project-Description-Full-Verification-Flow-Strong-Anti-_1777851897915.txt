📱 Step 3: Device Detection (IMPORTANT)

Inside the Mini App:

- Generate a strong device fingerprint (device_id) using:
  
  - browser/device info
  - screen size
  - system data
  - user agent

- Hash it securely before saving

- Save:
  
  - user_id
  - device_id (hashed)
  - IP address (secondary only)

---

🧠 Anti Multi-Account Logic

Rule:

- One device_id = one main account only

Detection must work even if:

- User installs Telegram clone
- Uses parallel space / dual apps
- Logs in with another account on same phone

👉 If same device_id appears again:

- This is multi-account abuse

🔴 If Duplicate Detected (VERY IMPORTANT FLOW)

  «"تم اكتشاف تعدد حسابات"»

  - banned OR not eligible

🟢 If Verification Successful

- Complete verification

Inside Bot:

- Show:
  
  «"تم فحص البوت بنجاح"»

- Show full welcome message and allow access
